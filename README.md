# The Alchemist's Cottage

A game.

### Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run dev
```

### Compiles and minifies for production
```
npm run build
```

### Locally previews build
```
npm run preview
```

### Runs unit test suite
```
npm run unit
```

### Runs end-to-end test suite
```
npm run e2e
```
