# The Alchemist's Cottage

A game.

## Made using Phaser

## First steps

This project requires [Node.js](https://nodejs.org) and [NPM.js](https://www.npmjs.com). 

* Install dependencies:

    ```
    npm install
    npm update
    ```

* Run the development server:

    ```
    npm start
    ```

    Open the browser at `http://127.0.0.1:8080`.
